Simple Calculator App (Flutter)

A fully functional Simple Calculator App built using Flutter.  
It performs all basic arithmetic operations and includes a Dark/Light theme switch for better user experience.

# Features :
1) Clean & Responsive UI 
2) Supports All Basic Operations: Addition, Subtraction, Multiplication, Division  
3) All Clear (AC) and Backspace functionality  
4) Decimal input support (.)
5) Dark/Light Theme Toggle (with icon button in AppBar)
